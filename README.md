# OpenAPI Specification with Swagger UI

This example demonstrates the relationship between OpenAPI and Swagger. It defines a simple API using the OpenAPI specification (a language-agnostic standard for describing APIs) and then uses Swagger UI (a tool from the Swagger project) to render this specification into interactive, human-readable documentation directly in the browser.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in any modern web browser.
